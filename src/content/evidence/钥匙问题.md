# P4890 解题代码详解（链式DP + 建模全过程）

[P4890 Never·island - 洛谷](https://www.luogu.com.cn/problem/solution/P4890)

------

# 一、整体思路（先建立全局认知）

本题流程分三步：

1. **离散化 + 扫描线建模**
   - 把区间拆成若干小段
   - 转换成「点权 + 边权 + 固定贡献」
2. **构建链式图**
   - 每个考察队是一个点
   - 区间贡献变成：
     - 点权（选一个点获得）
     - 边权（两个点都选才获得）
3. **链上DP + 背包合并**
   - 每条链独立DP
   - 最后用背包合并选 k 个点

------

# 二、变量说明（代码核心变量）

```cpp
int n, k;
```

- n：区间数量
- k：最多选的考察队数量

------

```cpp
vector<int> xs;
```

- 存所有端点（用于离散化）

------

```cpp
struct Seg {
    int l, r, id_l, id_r;
};
```

- l, r：原始端点
- id_l, id_r：离散化后的下标

------

```cpp
long long node_w[MAXN];
```

👉 点权数组

- node_w[i] 表示：选择第 i 个考察队获得的贡献

------

```cpp
vector<pair<int,int>> adj[MAXN];
```

👉 邻接表（存边）

- adj[u] = {v, weight}

------

```cpp
long long base_ans;
```

👉 必然贡献（情况4）

------

# 三、建模部分（最关键）

## 1️⃣ 离散化

```cpp
sort(xs.begin(), xs.end());
xs.erase(unique(xs.begin(), xs.end()), xs.end());
```

👉 把所有端点压缩到连续整数

------

## 2️⃣ 扫描每一小段区间

```cpp
for (int i = 1; i < xs.size(); i++) {
    int len = xs[i] - xs[i-1];
}
```

👉 每个小段长度 = 相邻离散点差值

------

## 3️⃣ 判断左右端点归属（核心）

假设：

```cpp
int L = belong_left[i];
int R = belong_right[i];
```

------

### 情况1：同一个队

```cpp
if (L == R) {
    node_w[L] += len;
}
```

------

### 情况2：不同队

```cpp
else {
    adj[L].push_back({R, len});
    adj[R].push_back({L, len});
}
```

------

### 情况3：都是起点

（代码里通常已经合并到情况1）

------

### 情况4：左终点右起点

```cpp
base_ans += len;
```

------

# 四、为什么是“链”

👉 每个点最多连接左右两个邻居
👉 不可能形成复杂图

所以：

```text
整个图 = 多条链
```

------

# 五、链上DP

## 状态定义

```cpp
dp[i][j][0/1]
```

含义：

| 变量 | 含义              |
| ---- | ----------------- |
| i    | 当前处理第 i 个点 |
| j    | 已选 j 个点       |
| 0    | 不选当前点        |
| 1    | 选当前点          |

------

## 初始化

```cpp
dp[0][0][0] = 0;
dp[0][0][1] = -INF;
```

------

## 转移

### 不选当前点

```cpp
dp[i][j][0] = max(dp[i-1][j][0], dp[i-1][j][1]);
```

------

### 选当前点

```cpp
dp[i][j][1] = max(
    dp[i-1][j-1][0] + node_w[i],
    dp[i-1][j-1][1] + node_w[i] + edge_w[i]
);
```

------

## edge_w[i] 含义

👉 i 和 i-1 之间的边权

------

# 六、链结果提取

每条链算出：

```cpp
f[x] = 选 x 个点的最大收益
```

------

# 七、多链合并（背包）

```cpp
for (每条链) {
    for (j = k → 0) {
        for (x = 0 → 链长度) {
            dp[j] = max(dp[j], dp[j-x] + f[x]);
        }
    }
}
```

------

# 八、最终答案

```cpp
answer = base_ans + dp[k];
```

------

# 九、容易出错的点（你之前卡的）

## ❌ 1. dp没初始化为 -INF

👉 会导致非法状态参与计算

------

## ❌ 2. 边权只在“两个点都选”时加

👉 很多人写错成只要有边就加

------

## ❌ 3. 没拆链直接DP

👉 会超复杂 + 错

------

## ❌ 4. 忘了 base_ans

👉 会少一大块答案

------

# 十、复杂度

- 建图：O(n log n)
- 链DP：O(n * k)
- 背包合并：O(n * k)

👉 总体：**O(n²)**（可过）

------

👉 本题就是：
