常见类型 INF 设定（竞赛常用）#

类型 位数 理论最大值 常用 INF 写法 说明
int 32bit 2,147,483,647 0x3f3f3f3f 最常见写法，可 memset
int 32bit 2,147,483,647 1e9 简单直观
long long 64bit 9.22e18 1LL<<60 最常用
long long 64bit 9.22e18 4e18 也常见
float 32bit ~3.4e38 1e30f 防止精度问题
double 64bit ~1.7e308 1e100 Floyd/概率DP常用
double 64bit ~1.7e308 1e18 若数据规模较小

为什么 0x3f3f3f3f 特别流行#

原因是可以这样初始化：

memset(dist, 0x3f, sizeof(dist));

结果每个字节：

00111111

拼成：

0x3f3f3f3f = 1061109567

足够大且安全。
