Hugo博客配置

October 31, 2025

hugo项目结构#

my-blog/
├── archetypes/         # 内容模板
├── assets/             # 需要 Hugo Pipes 处理的资源 (SCSS, JS)
├── content/            # Markdown 内容
├── layouts/            # 模板布局 (覆盖主题)
├── static/             # 静态文件 (图片, 字体)
├── themes/             # 主题目录
└── hugo.yaml           # 核心配置文件

快速生成并推送到github网页#

1.Windows系统#

编辑一个deploy.ps1文件(.bat 是 cmd 的脚本，而 .ps1 是 PowerShell 的脚本)实现自动化推送比如：

param (
    [string]$msg = $(Read-Host "请输入本次提交信息")
)

# 进入 Hugo 项目根目录
cd F:\myblog

# 生成 Hugo 网页（包含草稿）
hugo -D

# 进入 public/ 目录
cd public

# 添加所有修改
git add .

# 提交
git commit -m "$msg"

# 强制推送到 gh-pages 分支
git push -f origin gh-pages

# 返回项目根目录
cd ..

# 输出提示信息
Write-Host ""
Write-Host "======================================" -ForegroundColor Yellow
Write-Host "✅ 网站已成功部署！" -ForegroundColor Green
Write-Host "🌐 访问地址: https://caoxuan5211.github.io/" -ForegroundColor Cyan
Write-Host "======================================" -ForegroundColor Yellow
