这篇文章要解决什么

AGENTS.md-House 是一个专门收纳 AGENTS.md 模板、片段和规则经验的页面。

它不是只放一整份超长 AGENTS.md，而是把博客页面当成目录，把真正可复制的内容放进 md 代码块里。

后面补内容时，建议保持这个原则：

  • 外层标题负责检索和阅读。
  • 内层 md 代码块负责复制和复用。
  • 每个模板都写清楚适用场景、边界和验证方式。
  • 不确定是否通用的规则，先放到项目专用区，不要塞进 base。

## Git Rules
- 不自动 commit，除非用户明确要求。
- 不自动 push，除非用户明确要求。
- 不使用 `git reset --hard` 或危险回滚命令。
- commit 格式：

  - `fix(scope): 中文动词开头摘要`
