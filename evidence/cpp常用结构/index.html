1.unordered_set

用处

insert(),find(),erase(type)（删除指定元素）,clear()

2.memset

用法

void *memset(void *str, int c, size_t n)

参数

  • str -- 指向要填充的内存区域的指针。
  • c -- 要设置的值，通常是一个无符号字符。
  • n -- 要被设置为该值的字节数。

例子

memset(str,'
,7); memset(nums,0,sizeof(nums));
